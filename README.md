# Paradoxical SoundFont
Sources to build a paradoxical soundfont. It require [sfubar tool](https://github.com/freepats/tools.git) to build. To create the soundfont, type:
```
make
```
## Hierarchy

```
data/     -> Audacity sources files
waves/    -> WAV files
```
